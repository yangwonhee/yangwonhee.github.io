## 🌱 전체 코드

```py
def solution(s):
    # 입력값을 공백으로 나눠서 리스트(a)에 넣기.
    a = s.split()
    # 넣은 리스트의 str -> int로 변환
    a = [int(i) for i in a]
    # sort list int values
    a = sorted(a)
    # 리스트의 맨앞 맨뒤 숫자를 새로운 리스트(b)에 넣어서 int->str 변환.
    b = [str(a[0]), str(a[len(a)-1])]
    # 공백으로 합쳐주기
    answer = ' '.join(b)
    return answer
```
